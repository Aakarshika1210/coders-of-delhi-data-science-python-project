## Overview

This project simulates a **social network recommendation system**
similar to features used in platforms like LinkedIn or Facebook.

The system analyzes: - User friendships - Pages users like

Based on this information, it generates recommendations such as:

-   **People You May Know**
-   **Pages You May Like**

The project demonstrates basic **data cleaning, relationship analysis,
and recommendation logic using Python and JSON datasets**.

------------------------------------------------------------------------

## Project Workflow

### 1. Data Cleaning

Raw social network data is cleaned by: - Removing duplicate users and
pages - Fixing missing values - Standardizing friend relationships

Notebook: `02_data_cleaning.ipynb`

### 2. People You May Know

Friend recommendation system based on **mutual connections between
users**.

Notebook: `03_people_you_may_know.ipynb`

### 3. Pages You May Like

Page recommendations based on **pages liked by friends and shared
interests**.

Notebook: `pages_you_may_like.ipynb`
